Features

No feature tiers. No paywalls. Every feature works from day one — you only pay based on how many players your server has.

Exploit Protection

HWID Limiting

Restrict accounts per hardware device using CPU, GPU, motherboard, and HDD fingerprinting. Encrypted anti-spoof protection prevents bypasses.

IP Limiting & Geo-Block

Control connections per IP. Block or whitelist specific countries. Built-in VPN and proxy detection to stop bypass attempts.

DDoS Protection

Quick Start
Running in 3 Minutes

Download, configure your server IP, and start. It's that simple.

nexusfilter — setup
# Step 1 — Download NexusFilter
$ wget https://nexusfilter.io/releases/latest.zip

# Step 2 — Extract & Configure
$ unzip nexusfilter-v1.0.zip
$ nano config.json # Set server IP & port

# Step 3 — Start Protection
$ ./NexusFilter --host=YOUR_SERVER_IP --port=15779

✓ NexusFilter v1.0 — protecting your server!
✓ Web Panel → http://localhost:8080/panel
⚡ Gateway listening on port 15778
