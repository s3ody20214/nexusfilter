Quick Start
Up and Running in 3 Minutes

Download, configure, run. It's that simple.

nexusfilter — setup
# Step 1 — Download NexusFilter
$ wget https://nexusfilter.io/releases/latest.zip

# Step 2 — Extract & Configure
$ unzip nexusfilter-v1.0.zip
$ nano config.json # Set your server IP & port

# Step 3 — Start Protection
$ ./NexusFilter --host=YOUR_SERVER_IP --port=15779

✓ NexusFilter v1.0 is now protecting your server!
✓ Web Panel → https://localhost:8080/panel
⚡ Gateway listening on port 15778
