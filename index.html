Quick Start
Running in Under 3 Minutes

Download, configure your server IP, run. That's all.

nexusfilter — bash
# 1. Download NexusFilter
$ wget https://nexusfilter.io/releases/latest.zip
$ unzip nexusfilter-v1.0.zip -d /opt/nexusfilter

# 2. Set your server IP in config
$ nano config.json

# 3. Start — server is now protected
$ ./NexusFilter --host=192.168.1.100 --port=15779
