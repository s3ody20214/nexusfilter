What is NexusFilter?

NexusFilter is a high-performance packet filter (proxy) that sits between your Silkroad Online players and your game server. It inspects, filters, and secures every packet in real-time — protecting your server from exploits, bots, DDoS attacks, and abuse.

Quick Start

Get your server protected in under 3 minutes:

quick-start
# Start NexusFilter with minimal config ./NexusFilter --host=YOUR_SERVER_IP --port=15779 # With custom bind port ./NexusFilter --host=192.168.1.100 --port=15779 --bind-port=15778 # With web panel enabled ./NexusFilter --host=192.168.1.100 --port=15779 --panel=true --panel-port=8080
✅
That's it!

Your server is now protected. Access the web panel at http://localhost:8080/panel

Configuration File

The config.json file controls all NexusFilter settings. Here's a full example:

config.json
{ "server": { "host": "192.168.1.100", "gatewayPort": 15779, "bindGatewayPort": 15778, "bindAgentPort": 15882, "publicHost": "your-domain.com" }, "protection": { "exploitFilter": true, "ddosProtection": true, "maxConnectionsPerIP": 3, "maxHWIDAccounts": 2, "enableCaptcha": true, "captchaType": "image", "botDetection": true, "geoBlocking": { "enabled": false, "mode": "whitelist", "countries": ["EG", "TR", "SA"] } }, "packetDelays": { "enabled": true, "globalDelay": 100, "customDelays": { "0x7074": 500, "0x7045": 200 } }, "webPanel": { "enabled": true, "port": 8080, "username": "admin", "password": "change-me-now" }, "logging": { "packetLog": true, "exploitLog": true, "logPath": "./logs", "maxLogSize": "500MB" }, "license": { "key": "YOUR-LICENSE-KEY-HERE" } }

Exploit Protection

NexusFilter includes a comprehensive exploit database that covers all known Silkroad Online vulnerabilities. The database is auto-updated with new signatures.

Protected Exploits Include:

  • Gold Exploits — Duplicate gold, negative gold, overflow attacks
  • Item Exploits — Item duplication, invalid item creation, stack manipulation
  • Movement Exploits — Teleport hacks, speed hacks, wall walking
  • Trade Exploits — Stall/exchange manipulation, consignment abuse
  • Skill Exploits — Invalid skill usage, cooldown bypass, range exploits
  • Gateway Exploits — Login flood, character creation abuse, patch bypass

Configuration

config.json — protection section
"exploitFilter": true, // Enable/disable all exploit protection "exploitAction": "disconnect", // "log", "disconnect", "ban" "exploitBanDuration": 3600, // Ban duration in seconds (if action = ban) "autoUpdateSignatures": true // Auto-download new exploit signatures

HWID Limiting

Restrict the number of accounts per physical device using hardware fingerprinting. NexusFilter collects CPU serial, motherboard ID, GPU info, and HDD serial to create a unique device hash.

Configuration

config.json
"hwid": { "enabled": true, "maxAccountsPerHWID": 2, "action": "disconnect", // "log", "disconnect", "ban" "encryptionKey": "auto", // "auto" generates a random key "antiSpoof": true // Detect HWID spoofing attempts }
💡
How it works

NexusFilter injects a lightweight DLL into the game client that collects hardware identifiers. The data is encrypted before transmission to prevent tampering.

IP Limiting & Geo-Blocking

Control how many connections each IP address can make to your server, and optionally restrict access by country.

Configuration

config.json
"ipLimit": { "enabled": true, "maxConnectionsPerIP": 3, "whitelistedIPs": ["127.0.0.1"] }, "geoBlocking": { "enabled": true, "mode": "whitelist", // "whitelist" or "blacklist" "countries": ["EG", "TR", "SA", "DE"] }

DDoS Protection

Multi-layer DDoS mitigation that detects and blocks SYN flood, UDP flood, and application-layer attacks automatically.

config.json
"ddos": { "enabled": true, "maxPacketsPerSecond": 100, "connectionRateLimit": 10, // Max new connections per second per IP "banDuration": 300, // Auto-ban duration in seconds "synFloodProtection": true, "udpFloodProtection": true }

Bot Detection

Detect and block all known Silkroad bots including phBot, SBot, xBot, and custom bots.

Free Tier

Signature-based detection that identifies known bot patterns in packets.

Pro Tier

AI behavioral analysis that detects bots by analyzing movement patterns, action timing, and packet sequences — even for unknown or custom bots.

config.json
"botDetection": { "enabled": true, "signatureDetection": true, // Free: detect known bots "behavioralAnalysis": false, // Pro: AI-based detection "action": "disconnect", "notifyDiscord": false // Pro: send alert on detection }

Smart Packet Delays

Add intelligent delays per packet opcode to prevent speed hacks and timing-based exploits. Set a global delay or configure per-opcode.

config.json
"packetDelays": { "enabled": true, "globalDelay": 100, // ms — applied to all packets "customDelays": { "0x7074": 500, // Buy from NPC "0x7075": 500, // Sell to NPC "0x7045": 200, // Use skill "0x7021": 150, // Move "0x70A7": 1000 // Exchange confirm } }

FAQ

Is NexusFilter really free?

Yes! The free tier supports up to 50 players with full exploit protection, HWID limiting, IP limiting, DDoS protection, and basic captcha. No credit card required. No time limit.

Does it work with iSRO files?

NexusFilter is built for vSRO 1.188 files. iSRO-R support is planned for a future update.

Will it slow down my server?

No. NexusFilter adds less than 1ms of latency per packet. It's built with asynchronous processing and can handle 10,000+ concurrent connections.

Can players bypass the HWID limit?

NexusFilter uses encrypted multi-point fingerprinting (CPU + GPU + Motherboard + HDD). The anti-spoof feature detects common bypass tools. While no system is 100% unbreakable, NexusFilter makes bypassing significantly harder than alternatives.

How do I upgrade from Free to Pro?

Purchase a Pro license from the pricing page, enter the license key in your config.json, and restart NexusFilter. All Pro features unlock instantly.

Does it support Linux?

Yes! Full Linux support (Ubuntu 20.04+, Debian 11+, CentOS 8+). In fact, NexusFilter performs better on Linux with lower resource usage.
